Problem Description

The longest common Substring (LCSStr) problem is to find the longest string (or strings) that is a substring (or are substrings) of two strings. This problem required occupying consecutive positions within the original sequences.

For example, having the strings “AAAAAAABBAAB” and “ABAABBAB”, the Longest Common Substring is the string “AABBAB” with a length of 5.

Input

The input consists of a number of cases. For each case will come 2 strings ( 0 < length1, length <= 1000).

Output

For each case, print on a line with the length of the Longest Common Substring. Check the Sample Output for the presentation.
Sample Input

2
AAAAAAABBAAB
ABAABBAB
XMJYAUZ
MZJAWXU

Sample Output

Case 1: 5
Case 2: 1
